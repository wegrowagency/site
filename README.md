## ğŸ¨ Customization

### Changing Colors

Edit CSS variables in `css/style.css`:

```css
:root {
    --primary-green: #2E5D36;
    --brand-lime: #a7d84d;
    --brand-leaf: #22942c;
    /* ... and more */
}
```

### Updating Links

- **Calendly URL:** Update in `index.html` (search for `calendly.com`)
- **Email:** Update `mailto:` links in footer
- **LinkedIn:** Update the LinkedIn URL in the footer

### Replacing Images

Replace the image files in the `images/` folder with your own. Maintain the same filenames or update references in `index.html`.
